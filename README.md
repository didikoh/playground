# Hello World Backend with Next.js

A simple Next.js backend API ready for deployment on Vercel.

## API Endpoint

- `GET /api/hello` - Returns a Hello World JSON response

## Local Development

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Test the API:
```
http://localhost:3000/api/hello
```

## Deploy to Vercel

1. Push this code to a GitHub repository
2. Import the repository in Vercel
3. Deploy (Vercel auto-detects Next.js projects)

Your API will be available at: `https://your-project.vercel.app/api/hello`
